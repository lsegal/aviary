# Control Panel

This page maps the current web UI route structure into human-facing documentation sections.

## Top-Level Areas

### Overview

- Agent counts and state badges.
- Job counts and in-progress or failed indicators.
- Config validation and doctor-style issue reporting.
- Setup wizard when no agents exist yet.

### Chat

- Agent tabs and session switching.
- Streaming assistant responses and tool events.
- Message history, media attachments, and inline errors.
- Session creation and stop controls.

### Settings

- General runtime settings for server, models, browser, scheduler, and search.
- Agent editor for general fields, permissions, channels, files, and tasks.
- Providers and auth management.
- Skill enablement and session administration.
- Config save, reload, and backup restore actions.

### System

- Usage analytics.
- Job queue, schedules, logs, and compile attempts.
- Tool catalog and tool runner.
- Model catalog.
- Skill marketplace.
- Logs and daemon monitoring.
