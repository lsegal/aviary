# Getting Started

This page is a placeholder for the first-run path and should expand into the canonical onboarding flow.

## Scope To Cover

1. Install Aviary on macOS, Linux, and Windows.
2. Start the server and explain the default local HTTPS endpoint.
3. Retrieve and store the access token.
4. Log into the control panel.
5. Create the first agent and confirm a working chat session.

## Notes From The Current Product

- The server defaults to `https://localhost:16677`.
- The login screen is token-based.
- The setup wizard appears when no agents are configured.
- Configuration changes are applied live through the settings surface.
